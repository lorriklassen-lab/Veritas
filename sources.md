# Master Source List — Veritas Biblical Research Assistant

## Source Tiering System

All sources are categorized into three tiers:

- **🔵 PRIMARY** — Canonical sources that form the foundation. Answers should always reference these first.
- **🟢 STANDARD** — Widely respected sources for deeper study. Used regularly but not in every answer.
- **🟡 SUPPLEMENTAL** — Niche, denominational, or contextual sources. Used for specific topics or perspectives.

## 1. Bible Translations

### Primary Translations
| Translation | Abbreviation | Notes |
|---|---|---|
| New American Standard Bible (1995/2020) | NASB | Most literal word-for-word translation. Preferred for word-study and precise exegesis. |
| English Standard Version | ESV | Excellent balance of formal equivalence and readability. Standard for many seminaries. |
| Christian Standard Bible | CSB | Modern formal-equivalence translation by evangelical scholars. |
| New International Version (2011) | NIV | Most widely read modern translation. More dynamic equivalence but widely accepted. |

| Translation | Abbreviation | Notes |
|---|---|---|

| New King James Version | NKJV | Modern update preserving KJV textual basis. |
| New Revised Standard Version (Updated) | NRSVue | Ecumenical translation, preferred in academic mainline circles. |
| Lexham English Bible | LEB | Useful for its consistent translation of Greek tenses and Hebrew constructs. |

|---|---|---|

## 12. Source Selection Principles

When selecting which sources to use for a given answer:

1. **Start with primary** — Consult primary tier sources first (BDAG, NASB/ESV, NICNT/NICOT, NIDNTTE).
2. **Match the source to the question** — Historical questions need background sources; word-study questions need lexicons; theological questions need systematic theology.
3. **Represent the spectrum** — For controversial topics, cite views from at least 2-3 credible positions.
4. **Prefer recent scholarship** — All else equal, prefer more recent works that incorporate latest discoveries.
5. **Cite the best accessible source** — Not every answer needs BDAG; use Strong's/Vine's for lay-level answers when appropriate.
6. **No prosperity gospel sources** — Exclude known prosperity gospel, word-of-faith, and hyper-charismatic writers (e.g., Kenneth Copeland, Creflo Dollar, Joel Osteen, Joyce Meyer, Benny Hinn).
7. **No speculative sources** — Exclude sources that present speculation as fact (e.g., Dan Brown style, "Bible code" literature, Christ myth theorists who misrepresent scholarship).
