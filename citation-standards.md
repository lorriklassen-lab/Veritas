# Citation Standards — Veritas Biblical Research Assistant

## 1. Core Principles

Every answer produced by Veritas **must cite its sources**. Users should always be able to verify the answer's claims. Our credibility depends on transparency.

### Principles
1. **Every factual claim needs a citation** — If you state something (a passage meaning, a historical fact, a lexical definition), back it up.
2. **Cite the best source, not just any source** — Use primary-tier sources first, then standard.
3. **Be specific** — "See BDAG" is not a citation. "BDAG, s.v. 'ἀγάπη,' meaning 1b" is.
4. **Let the user dig deeper** — Every citation is an invitation for the user to explore further.

---

## 2. Citation Format — Bible Passages

### Standard Format
```
Book Chapter:Verse (Translation Abbreviation)
```

**Examples:**
- John 3:16 (NASB)
- Genesis 1:1–2:3 (ESV)
- Romans 9:14–18 (NIV)
- Psalm 23:1–6 (CSB)

### When Multiple Translations Are Used
```
Book Chapter:Verse (Translation1, Translation2)
```

**Example:**
- Ephesians 2:8–9 (NASB, ESV)

### Parallel Passages (Synoptics / Parallel accounts)
```
Matthew 5:3 // Luke 6:20
```
Use `//` to indicate synoptic or parallel passages.

### Range Format
- Verse span: `Romans 8:31–39`
- Chapter span: `Genesis 1:1–2:3`
- Broken span (verses within a larger passage): `John 14:6 (cf. 14:1–14)`

## 3. Citation Format — Lexicons & Original-Language Tools

### Lexicon Entries
```
[Lexicon Abbreviation], s.v. "[Word in Greek/Hebrew]," [Meaning/Definition Reference].
```

**Examples:**
- BDAG, s.v. "λόγος," meaning 1b — "a unit of speech, word, saying"
- HALOT, s.v. "חֶסֶד," definition 2 — "loyalty, faithfulness, steadfast love"
- BDB, s.v. "שָׁלוֹם," meaning 2 — "completeness, soundness, welfare, peace"

### Greek/Hebrew Words in Answers
- Greek: Use Greek script with transliteration in parentheses on first use: ἀγάπη (agapē)
- Hebrew: Use Hebrew script with transliteration on first use: חֶסֶד (ḥesed)
- Include Strong's number in parentheses for lay-level answers: ἀγάπη (agapē, G26)

### Grammatical Citations
```
[Author], [Grammar Title], [Page/Section].
```

**Examples:**
- Wallace, *Greek Grammar Beyond the Basics*, 287–290.
- Waltke & O'Connor, *Introduction to Biblical Hebrew Syntax*, §14.3a.

## 4. Citation Format — Commentaries

### Standard Format
```
[Author], [Commentary Title], [Volume if multi-vol], [Page or Section Reference].
```

**Examples:**
- Fee, *The First Epistle to the Corinthians* (NICNT), 123–125.
- Moo, *The Epistle to the Romans* (NICNT), 567–569.
- Waltke, *Genesis: A Commentary*, 234.
- Longman, *Psalms* (TOTC), 89.

## 5. Citation Format — Reference Works

### Standard Format
```
[Work Title], [Editor(s)], [Edition if applicable], [Entry/Page].
```

**Examples:**
- *NIDNTTE*, Silva (ed.), vol. 2, 345–348, s.v. "δικαιοσύνη."
- *Dictionary of Jesus and the Gospels* (IVP), Green, McKnight, Marshall (eds.), 2nd ed., 456–460.
- *Eerdmans Dictionary of the Bible*, Freedman (ed.), s.v. "Tabernacle."

---

## 6. Citation Format — Books & Articles

### Books
```
[Author], [Book Title] ([Publisher], [Year]), [Page].
```

**Examples:**
- Carson, *Exegetical Fallacies* (Baker, 1996), 89–91.
- Wright, *The Resurrection of the Son of God* (Fortress, 2003), 610–615.
- Keener, *The IVP Bible Background Commentary: New Testament* (IVP, 2nd ed., 2014), 210.

### Journal Articles
```
[Author], "[Article Title]," [Journal Name] [Volume]([Year]): [Pages].
```

**Example:**
- Bauckham, "The Worship of Jesus in Apocalyptic Christianity," *NTS* 27(1981): 322–341.

---

## 7. Citation Format — Historical Sources

### Ancient Sources
```
[Author], [Work Title], [Book.Chapter.Section or standard reference].
```

**Examples:**
- Josephus, *Antiquities*, 18.3.3.
- Eusebius, *Ecclesiastical History*, 3.39.
- Augustine, *Confessions*, 7.9.

### Creeds & Confessions
```
[Document Title], [Chapter/Article/Section].
```

**Examples:**
- *Nicene Creed* (AD 325/381).
- *Westminster Confession of Faith*, Chapter 2, §1.
- *Catechism of the Catholic Church*, §460.

## 8. Citation Placement in Answers

### Inline (Preferred for concise answers)
When a citation is brief, include it inline in parentheses.

> The Greek word for "love" in John 3:16 is ἀγάπη (agapē), which BDAG defines as "the quality of warm regard for and interest in another" (BDAG, s.v. "ἀγάπη," def. 1a).

### Footnote / Endnote Style (For longer answers)
For detailed answers, use numbered footnotes or endnotes.

> The term "propitiation" in Romans 3:25 conveys the idea of Christ's sacrifice satisfying God's righteous wrath against sin.⁴

With footnote:
> ⁴ Leon Morris, *The Apostolic Preaching of the Cross* (Eerdmans, 3rd ed., 1965), 181–185. Cf. Schreiner, *Romans* (BECNT), 189.

### References at End (For summarized answers)
For short summaries, list key references at the end.

> **Key Sources:** NASB; BDAG, s.v. "χάρις"; Schreiner, *Romans* (BECNT), 155–157.

## 9. Handling Contradictory Sources

When scholars disagree:

1. **Cite both positions fairly.**
2. **Identify which is the majority view** if one clearly is.
3. **Use neutral framing:**
   - "Most commentators (e.g., Moo, Schreiner) understand this passage to mean X. However, a minority view (e.g., Dunn, Stuhlmacher) argues for Y."
   - "There is debate on this point. Fee argues X, while Thiselton argues Y."

4. **Never present a disputed interpretation as settled fact** unless there is a strong consensus.

## 10. Source Quality Markers

When a source is cited, optionally indicate its tier:

| Marker | Meaning | Example |
|---|---|---|
| (P) | Primary Tier | BDAG, NICNT, NASB |
| (S) | Standard Tier | WBC, IVP Dictionaries |
| (Sup) | Supplemental Tier | Individual denominational sources |

Used sparingly — only when the user needs to assess weight of evidence.

---

## 11. What NOT to Do

| Don't | Why |
|---|---|
| Cite without reference | "As scholars agree" — which scholars? |
| Cite a translation as a source for a lexical claim | "NASB says..." for word meaning — use lexicons |
| Cite a secondary source when a primary exists | Cite BDAG directly, not a book quoting BDAG |
| Over-cite obvious claims | "John 3:16 (NASB)" is enough — no need to also cite a commentary saying it's famous |
| Cite by URL alone | Always include author/work/page |
| List "Google" or "Wikipedia" as a source | These are not authoritative — trace to the original |
| Cite prosperity gospel / speculative sources | Excluded by policy |

---

## 12. Translation Handling

When comparing translations or citing the original text:

1. **Cite the translation you are quoting** — always include the abbreviation.
2. **Provide the Greek/Hebrew for key terms** — when a word is theologically significant, include the original.
3. **Acknowledge translation differences:**
   - "The ESV renders this as X, while the NIV translates Y (more dynamic equivalence)."
   - "The KJV's use of 'charity' (1 Corinthians 13) translates ἀγάπη (agapē), which modern translations render as 'love.'"
4. **Never say "the Bible says" when you mean "the [Translation] says"** — be transparent.

## 13. Example Answer with Citations

> **Question:** What does "faith" mean in Hebrews 11:1?
>
> **Answer:**
> Hebrews 11:1 (NASB) defines faith as "the assurance of things hoped for, the conviction of things not seen." The key Greek term is πίστις (pistis), which in this context carries the sense of "conviction, confidence, or assurance" about unseen realities (BDAG, s.v. "πίστις," def. 2b). The NASB's rendering of ὑπόστασις (hypostasis) as "assurance" is preferred by many commentators (Lane, *Hebrews* [WBC], 327–329), though some translate it as "substance" or "realization" (Ellingworth, *Hebrews* [NIGTC], 567). In either case, the emphasis is on faith as a present reality that makes the future certain and the unseen visible.
>
> **Sources:** NASB; BDAG, s.v. "πίστις"; BDAG, s.v. "ὑπόστασις"; Lane, *Hebrews* (WBC), 327–329; Ellingworth, *Hebrews* (NIGTC), 567.
